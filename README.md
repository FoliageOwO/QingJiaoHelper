# QingJiaoHelper

## 使用方法

1. 安装油猴脚本插件
2. 在 [Greasy Fork - QingJiaoHelper](https://greasyfork.org/zh-CN/scripts/452984-qingjiaohelper) 中安装脚本
3. 详细用法见脚本主页

## 构建

> [!NOTE]
> 你需要装有 `Node.js 18+` 的环境

1. 克隆仓库到本地

```bash
$ git clone https://github.com/WindLeaf233/QingJiaoHelper.git
```

2. 安装包

```bash
$ cd QingJiaoHelper
$ npm i # 或者用你喜欢的包管理器，例如 `cnpm`
```

3. 开始构建

```bash
$ npm run gen
```

4. 构建完成的产物为 `./dist.js`
